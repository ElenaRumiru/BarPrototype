# 🍷 Bar Prototype

Прототип механики управления баром — casual-симулятор, где вы управляете сменой, обслуживаете гостей и зарабатываете деньги.

**[▶ Играть в демо](https://elenarumiru.github.io/BarPrototype/)**

## Об игре

Интерактивный прототип для проработки ключевых механик бар-менеджера. Игрок управляет сменой: рассаживает гостей по диванам, выполняет заказы на напитки, поддерживает социальное взаимодействие и удерживает уходящих посетителей.

## Геймплей

**Смена** — ограниченное время, за которое нужно обслужить максимум гостей на 5 диванах.

**Заказы бара** 🍷🍹🍸🍺 — готовьте напитки вовремя, пока таймер не истёк.

**Социальные навыки** ❤️💬🔥 — реагируйте на запросы гостей для повышения лояльности.

**Удержание** ⏰ — мини-механика, чтобы не дать гостю уйти раньше времени.

**Три класса гостей** — от нищего 💸 до бизнесмена 💼, каждый со своим временем визита и лимитом удержаний.

## Настройки баланса

Прототип включает панель конфигурации прямо в интерфейсе — можно на лету менять время смены, цены напитков, кулдауны, штрафы, параметры классов гостей и всё остальное без правки кода.

## Технологии

Vanilla JavaScript (ES Modules), HTML, CSS. Без фреймворков и библиотек. Шрифт Nunito.

## Запуск локально

```bash
git clone https://github.com/ElenaRumiru/BarPrototype.git
cd BarPrototype
npx serve .
```

ES Modules требуют HTTP-сервера — через `file://` не откроется.
