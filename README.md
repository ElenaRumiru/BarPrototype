 # 🍷 Bar Prototype                                                                                            
                                                                                                                  > Прототип механики управления баром — casual-симулятор, где вы управляете сменой в баре, обслуживаете гостей   и зарабатываете деньги.                                                                                       
                                                                                                                  **[▶ Играть в демо](https://elenarumiru.github.io/BarPrototype/)**                                                                                                                                                            
  ---                                                                                                                                                                                                                           
  ## Об игре                                                                                                                                                                                                                    
  Интерактивный прототип для проработки ключевых механик бар-менеджера. Игрок управляет сменой: рассаживает       гостей по диванам, выполняет заказы на напитки, поддерживает социальное взаимодействие и удерживает уходящих  
  посетителей.                                                                                                  
                                                                                                                
  ### Основной игровой цикл                                                                                        
  - **Смена** — ограниченное время, за которое нужно обслужить максимум гостей                                    - **5 диванов** — гости приходят, заказывают напитки и требуют внимания                                       
  - **Заказы бара** 🍷🍹🍸🍺 — готовьте напитки вовремя, пока таймер не истёк                                   
  - **Социальные навыки** ❤️💬🔥 — реагируйте на запросы гостей для повышения лояльности                         
  - **Удержание** ⏰ — мини-механика, чтобы не дать гостю уйти раньше времени                                   
                                                                                                                  ### Типы гостей                                                                                               
                                                                                                                  | Класс | Особенности |                                                                                         |-------|-------------|
  | 💸 Нищий | Короткое время визита, мало удержаний |                                                          
  | 👔 Средний класс | Сбалансированные параметры |                                                             
  | 💼 Бизнесмен | Долгий визит, больше удержаний |                                                             
                                                                                                                  ## Настройки баланса                                                                                          
                                                                                                                  Прототип включает панель конфигурации для тонкой настройки всех параметров без правки кода:                      
  - Время смены, задержки появления гостей                                                                        - Таймеры заказов и социальных запросов
  - Время приготовления и цены напитков                                                                           - Параметры классов гостей (время визита, лимит удержаний, вес появления)
  - Кулдауны и штрафы                                                                                           
                                                                                                                  ## Технологии                                                                                                 
                                                                                                                  - Vanilla JavaScript (ES Modules)                                                                               - HTML + CSS (без фреймворков и библиотек)
  - Шрифт [Nunito](https://fonts.google.com/specimen/Nunito)                                                    
  - Деплой на GitHub Pages
